# CI/CD Pipeline Basics

## Objective

This project demonstrates a basic Continuous Integration (CI)
pipeline using GitHub Actions.

## Technologies

- Python
- Flask
- Pytest
- Docker
- GitHub Actions

## Pipeline Stages

1. Checkout Code
2. Setup Python
3. Install Dependencies
4. Run Tests
5. Build Docker Image

## How to Run

pip install -r requirements.txt

python app.py

Visit:

http://localhost:5000

Run tests:

pytest
